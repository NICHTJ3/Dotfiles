## Running

```sh
git clone https://github.com/NICHTJ3/Dotfiles ~/Dotfiles
cd ~/Dotfiles

dotsy --help
# dotsy profile list > To list available profiles
# dotsy profile install work-mac > To install the work-mac profile
```
